User Guide

Welcome to the Nexus Project Management System! This guide will walk you through the essential features of our platform.

Dashboard

The dashboard is your central hub for managing files. Here you can upload, download, and delete files with ease.

File Upload

To upload a file, simply click the "Upload" button and select the file you want to store. The file will be securely uploaded to your account.

File Download

To download a file, click the download icon next to the file name. The file will be saved to your local machine.

CLI Usage

The Nexus CLI allows you to interact with the file management system from your terminal. To get started, you'll need to download the client and configure it with your credentials.

Configuration

First, run the configure command and provide your server URL, username, and password:

python nexus_client.py configure

Uploading Files

To upload a file, use the upload command followed by the file path:

python nexus_client.py upload /path/to/your/file.txt

Downloading Files

To download a file, use the download command followed by the file name:

python nexus_client.py download your-file.txt
